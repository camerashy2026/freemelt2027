## 🔑 首次使用

页面打开后会自动弹出 API Key 设置框：

1. 打开 [console.anthropic.com](https://console.anthropic.com) 注册/登录
2. 左侧 **API Keys** → **Create Key**
3. 复制 `sk-ant-...` 开头的密钥，粘贴到弹窗中保存

密钥仅保存在本地浏览器，不会上传到任何服务器。新用户有免费额度，每次搜索约消耗 $0.01–0.03。

## 功能特性

- **七路并行 AI 搜索**：obplib 官方 · GitHub 仓库 · EBM 论文 · Gyroid 实验 · 切片软件生态 · 新开源工具 · 综合分析
- **结构化周报**：包含核心发现、技术空白评估、行动建议、信息来源
- **资源监测中心**：完整追踪所有 GitLab/GitHub/PyPI 仓库 + 商业软件
- **时间线积累**：历史记录持久保存，形成研究知识库
- **Markdown 导出**：一键导出所有历史周报
- **零依赖**：纯 HTML，无需服务器，无需数据库
