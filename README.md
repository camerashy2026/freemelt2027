# FREEMELT 晶格 2027 🔬

**Freemelt E-PBF 晶格结构打印 · 周度研究追踪模块**

> 自动搜索 Freemelt 晶格打印领域每周最新进展，AI 生成结构化研究摘要，历史记录永久保存。

---

## 🚀 在线访问

部署后访问：`https://<你的GitHub用户名>.github.io/freemelt2027/`

---

## 功能特性

- **每周 AI 搜索**：点击一次，自动调用 Claude API 搜索最新论文、代码库、实验结果
- **三路并行搜索**：OBP/obplib 动态 · EBM 晶格论文 · 工具与开源项目
- **结构化周报**：每份报告包含核心发现、行动建议、信息来源、进展评级
- **时间线积累**：所有历史记录持久保存，形成研究知识库
- **零依赖部署**：纯 HTML，无需服务器，无需数据库
